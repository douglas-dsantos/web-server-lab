01
Linux & Terminal
⏱ 2 meses📍 Agora — durante o ADS🎯 Base obrigatória
▼
Cursos
LinuxTips Essentials — Linux
Fundamentos de Linux com o Jeferson Fernando, mesmo instrutor da certificação paga. Terminal, permissões, processos, scripts de shell. Gratuito na plataforma.
gratuitolinuxtips.io
YouTube — Canal LINUXtips
Centenas de horas gratuitas no YouTube. Acompanha o canal e assiste as playlists de Linux para iniciantes. Complementa os Essentials.
gratuitoyoutube.com/@linuxtips
Projeto Prático
Lab pessoal no VirtualBox
Instala Ubuntu 22.04 em uma VM. Pratica no terminal todo dia: criação de pastas, usuários, permissões, gerenciamento de processos. Documenta cada comando novo num README.
GitHub: linux-homelab
Servidor web local (Apache/Nginx)
Configura e sobe um servidor HTTP na VM. Serve uma página HTML simples. Ajusta permissões de arquivos e analisa logs de acesso. Primeiro projeto real no portfólio.
GitHub: web-server-lab
Meta da fase
Navegar no terminal com confiança
Sem depender do Google para comandos básicos. Conseguir criar, mover, editar e deletar arquivos; gerenciar permissões e processos; configurar um servidor simples do zero.
02
Python Básico & Git
⏱ 2 meses📍 Meses 3–4🎯 Automação e versionamento
▼
Cursos
CS50P — Introdução ao Python (Harvard)
Lógica de programação em Python. Totalmente gratuito. Não é Python avançado — é raciocínio lógico aplicado. Ideal para automação em DevOps. Sem necessidade de experiência anterior.
gratuitocs50.harvard.edu/python
GitHub Skills — Git e GitHub
Curso interativo oficial do GitHub. Aprende versionamento na prática, direto no repositório. Cobre branches, pull requests, merge e fluxo de trabalho profissional.
gratuitogithub.com/skills
Projetos Práticos
Scripts de automação com Python
Escreve scripts que automatizam tarefas do dia a dia: renomear arquivos em lote, organizar pastas, fazer backup de diretórios. Tudo versionado no GitHub desde o primeiro commit.
GitHub: python-automation-scripts
Portfólio no GitHub — início
Todos os exercícios do CS50P e do lab Linux vão para o GitHub. README bem escrito em cada repositório. Histórico de commits consistente já é portfólio.
GitHub: profile + repositórios organizados
Redes (paralelo)
Cisco Networking Academy — Fundamentos
TCP/IP, DNS, HTTP, firewall e VPN. Módulos introdutórios gratuitos. Não precisa terminar tudo — foca nos primeiros módulos que cobrem o essencial para DevOps.
gratuitonetacad.com
03
Docker & Containers
⏱ 2 meses📍 Meses 5–6🎯 Coração do DevOps moderno
▼
Cursos
LinuxTips Essentials — Docker
Módulo de Docker gratuito da LinuxTips. Com a base de Linux que você terá, esse conteúdo vai fazer muito sentido. Cobre criação de imagens, containers, volumes e redes.
gratuitolinuxtips.io
Play with Docker — Labs online
Ambiente online gratuito da Docker Inc. para praticar sem precisar instalar nada localmente. Ótimo para experimentar antes de configurar no seu próprio ambiente.
gratuitolabs.play-with-docker.com
Projetos Práticos
Containerizar uma aplicação Python
Pega um dos seus scripts Python da fase anterior e cria um Dockerfile para ele. Faz build da imagem, roda o container, testa. Documenta o processo no README.
GitHub: python-docker-app
Docker Compose com múltiplos serviços
Sobe um ambiente com app + banco de dados usando Docker Compose. Um exemplo clássico é uma API simples + PostgreSQL. Esse projeto já simula ambiente de produção real.
GitHub: docker-compose-lab
Meta da fase
Criar e gerenciar containers sem tutorial
Conseguir criar imagens, subir containers, configurar redes e volumes, e usar Docker Compose para orquestrar múltiplos serviços sem precisar copiar comandos da internet.
04
AWS & Cloud Foundations
⏱ 2 meses📍 Meses 7–8🎯 Primeira certificação
▼
Cursos
AWS Cloud Practitioner Essentials
Curso oficial gratuito da Amazon no AWS Skill Builder. Cobre todos os conceitos do exame CLF-C02: serviços, segurança, preços e arquitetura básica. Suficiente para a prova com estudo complementar.
gratuitoskillbuilder.aws
Simulados — Whizlabs ou Udemy
Antes de agendar a prova, faz simulados. Pacotes no Udemy custam ~R$ 30 em promoção. Taxa de aprovação sobe muito com essa preparação. Não pula essa etapa.
~R$ 30udemy.com
Projetos Práticos
Site estático no S3 + CloudFront + HTTPS
Sobe seu portfólio pessoal no S3. Configura CloudFront como CDN. Adiciona certificado SSL via ACM. Custo próximo de zero no free tier da AWS. Resultado: portfólio online real.
GitHub: portfolio-aws
IAM com princípio de least privilege
Cria usuários, grupos e políticas customizadas no IAM. Habilita MFA em todas as contas. Documenta as decisões de cada política. Demonstra pensamento de segurança no portfólio.
GitHub: iam-policies-lab
Certificação
★
AWS Certified Cloud Practitioner (CLF-C02)
Prova online, ~$100 (~R$ 550). Primeira certificação oficial no currículo. Reconhecida globalmente. Valida que você entende cloud de verdade.
05
DevOps Professional
⏱ 12 meses📍 A partir do mês 9🎯 Empregabilidade júnior
▼
Treinamento Principal
Certificação DevOps Professional — LINUXtips
12 meses, +300h. Instrutores de mercado real: Jeferson Fernando (fundador LinuxTips), Camilla Martins (SRE Storyblok), Rafael Gomes (DevOps Tech Lead). Cobre Terraform, AWS avançado, Kubernetes, GitHub Actions, Prometheus, Grafana e EKS. Em português, com aulas ao vivo e comunidade no Discord. O melhor investimento identificado para DevOps no Brasil.
investimentolinuxtips.ioportuguês
Projetos Práticos da Fase
Pipeline CI/CD completo com GitHub Actions
Cria um pipeline que faz build, testa e faz deploy automático de uma aplicação containerizada na AWS. Esse projeto sozinho já impressiona recrutadores.
GitHub: cicd-pipeline-aws
Infraestrutura como código com Terraform
Provisiona uma VPC com subnets, security groups e instâncias EC2 usando Terraform. Documenta cada decisão. Demonstra que você pensa em infra como código, não como cliques no console.
GitHub: terraform-aws-infra
Cluster Kubernetes com monitoramento
Sobe um cluster Kubernetes (EKS ou local com kind), faz deploy de uma aplicação e configura Prometheus + Grafana para monitoramento. Dashboard funcional de verdade.
GitHub: k8s-monitoring-lab
Certificações da Fase
★
HashiCorp Terraform Associate
Certificação reconhecida de IaC. ~$70 (~R$ 380). Terraform é a ferramenta padrão do mercado para infraestrutura como código. Muito valorizada em ofertas de emprego.
★
CKA — Certified Kubernetes Administrator
A certificação de Kubernetes mais respeitada do mercado. ~$395 (~R$ 2.100). Quem tem CKA no currículo raramente fica desempregado em DevOps. Meta de longo prazo desta fase.

06
Upgrades & Especialização
⏱ Contínuo📍 Após emprego na área🎯 Crescimento de carreira
▼
Caminhos possíveis
Platform Engineering
Construir plataformas internas que aceleram times de desenvolvimento. Próximo passo natural após DevOps sólido. Alta demanda em empresas de médio e grande porte.
Site Reliability Engineering (SRE)
Foco em confiabilidade, uptime e observabilidade. Carreira bem estabelecida em bancos, fintechs e big techs como Nubank, iFood e PicPay. Salários de R$ 18–30k+.
Multi-cloud: Azure AZ-104 + GCP Associate
Após dominar AWS, adicionar Azure e GCP expande muito as oportunidades. Profissionais multi-cloud são os mais bem pagos da área de infraestrutura cloud.
DevSecOps
Integração de segurança no pipeline DevOps. Perfil raríssimo e muito disputado. Combina o que você aprendeu em DevOps com princípios de segurança de aplicações.
